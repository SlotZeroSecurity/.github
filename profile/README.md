## Audits

**Slot Zero Security** provides focused security reviews for Solana programs, with emphasis on protocol logic, account validation, PDA authority design, CPI safety, vault accounting, state transitions, and upgrade authority risks.

We review both **native Rust Solana programs** and **Anchor-based codebases**, helping teams catch security issues before mainnet exposure.
